## 🛠️ Installation & Setup

```bash
# Install with research features
pip install freya[research]

# Enable research modes
freya config research --enabled

# Start with chat exchange
freya serve --research --exchange
```

### Research Configuration

```yaml
research:
  modes:
    assisted:
      enabled: true
      guidance_level: high
      suggestions: true
    autonomous:
      enabled: true
      max_duration: 7200
      quality_threshold: 0.85

  collaboration:
    multi_user: true
    shared_spaces: true
    peer_review: true

  analytics:
    dashboards: true
    visualizations: true
    reporting: true

chat_exchange:
  multi_user: true
  persistence: true
  search: true
  export: true
```

## 🚀 Usage Examples

### Research Workflows

```bash
# Start assisted research
freya research assisted --topic "Quantum Computing" --depth comprehensive

# Launch autonomous research
freya research autonomous --objective "Market Analysis 2026" --deadline 2026-12-31

# Join collaborative research
freya research collaborate --session quantum-research-001 --role analyst
```

### Chat Exchange

```bash
# Create shared conversation
freya chat exchange --create "AI Ethics Discussion" --participants 5

# Join existing exchange
freya chat exchange --join ethics-discussion-001

# Search conversations
freya chat search --query "machine learning ethics" --exchange
```
