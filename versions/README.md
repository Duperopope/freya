## Versioning Scheme

Freya uses semantic versioning:

- **v0.x**: Early development and foundation
- **v1.x**: TUI (Terminal User Interface) development
- **v2.x**: Modern web interface and advanced features

## How to Use

Each versioned README reflects the state of the project at that specific version, including:

- Features available at that time
- Installation instructions
- Usage examples
- Known limitations

To see what changed between versions, compare the files or refer to the interactive mindmap.
