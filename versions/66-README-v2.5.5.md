## 🛠️ Installation & Setup

```bash
# Install with cyber security features
pip install freya[cyber]

# Enable CyberAgent
freya config cyber --enabled

# Start with full security suite
freya serve --cyber --benchmarks
```

### Security Configuration

```yaml
cyber:
  enabled: true
  threat_detection:
    code_analysis: true
    data_protection: true
    real_time_monitoring: true

  compliance:
    standards: ["GDPR", "ISO27001", "NIST"]
    automated_checking: true
    reporting: true

  incident_response:
    automated: true
    containment: true
    forensics: true

benchmarks:
  external:
    enabled: true
    standards: ["GLUE", "MMLU", "ARC"]
    frequency: "daily"
    reporting: true

  custom:
    enabled: true
    metrics: ["accuracy", "speed", "efficiency"]
    thresholds:
      accuracy: 0.85
      speed: 1000 # tokens/second
```

## 🚀 Usage Examples

### Cyber Security Operations

```bash
# Run security analysis
freya cyber analyze --target codebase --comprehensive

# Monitor threats in real-time
freya cyber monitor --real-time --alerts

# Generate compliance report
freya cyber compliance --standard GDPR --report
```

### Benchmarking

```bash
# Run external benchmarks
freya bench external --standards GLUE,MMLU --comprehensive

# Compare model performance
freya bench compare --models gpt-4,llama2,mistral --metrics accuracy,speed

# Generate certification report
freya bench certify --model llama2 --standard MMLU
```

### Agent Selection

```bash
# Optimize agent selection
freya agents optimize --task "code_review" --criteria accuracy,speed

# Monitor agent performance
freya agents monitor --real-time --dashboard

# Update agent rankings
freya agents rank --update --historical-data
```
