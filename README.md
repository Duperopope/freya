## 🆕 What's New

### Recent Major Updates

Since version 0.9, Freya has undergone significant evolution with multiple major milestones. The project transitioned from a text-based interface to a modern web application, introduced advanced research capabilities, and implemented comprehensive multi-agent orchestration features.

**Research & Collaboration Enhancements (v0.9.0 - v0.10.0)**: Extended research modes with assisted and autonomous workflows, enhanced chat exchange capabilities for multi-user conversations, and improved collaborative research tools.

**Advanced Agent Systems (v0.11.0)**: Integration of CyberAgent for sophisticated multi-agent coordination, intelligent agent selection algorithms, and external benchmarking capabilities for performance evaluation.

**Modern Web Interface Revolution (v0.0.6)**: Complete architectural shift from TUI to a React 18 + TypeScript web application with FastAPI backend, featuring six comprehensive pages (Chat, Bench, BMAD Studio, Settings, Files, Watch) and real-time WebSocket communication.

**Hybrid Routing & Multi-Provider Support (v2.1+)**: Implementation of intelligent LLM routing between local Ollama models and remote OpenAI providers, with comprehensive provider management and routing optimization.

**Enhanced BMAD Pipeline (v2.4+)**: Advanced multi-agent decision orchestration with seven specialized agents, autonomous operation modes, real-time logging, and self-healing pipeline capabilities.

**Cyber Security & Monitoring (v2.5+)**: AI-powered threat analysis integration, comprehensive security monitoring, and advanced external benchmarking against industry standards.

The current version represents a mature, enterprise-ready multi-agent orchestrator with modern web interfaces, hybrid LLM routing, and comprehensive research and development automation capabilities.

## 📋 Overview

Freya is a **BMAD-aligned multi-agent orchestrator** for local LLMs, featuring hybrid routing, real-time collaboration, and comprehensive benchmarking. This repository contains the complete evolution of Freya from its initial release to the current version.

### 🎯 Key Features

- **Multi-Agent Orchestration**: 7 specialized agents in coordinated BMAD pipelines
- **Hybrid LLM Routing**: Intelligent selection between local (Ollama) and remote (OpenAI) models
- **Real-Time Chat**: Multiple conversation modes with autonomous operation
- **Comprehensive Benchmarking**: Performance testing with external benchmarks
- **Modern Web Interface**: React 18 + TypeScript with real-time updates
- **Cyber Security Integration**: AI-powered threat analysis and monitoring

## 🚀 Quick Start

```bash
# Clone and install
git clone https://github.com/your-repo/freya.git
cd freya
pip install -r requirements.txt
cd web && npm install && cd ..

# Start the application
python -m freya
```

For detailed installation instructions, see the [latest version documentation](versions/59-README-v0.9.04.md#-installation--setup).
