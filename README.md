## 🆕 What's New

### Recent Major Updates

Since version 0.0.9, Freya has undergone significant evolution with multiple major milestones. The project transitioned from a text-based interface to a modern web application, introduced advanced research capabilities, and implemented comprehensive multi-agent orchestration features.

**Modern Web Interface Revolution (v0.3.0)**: Complete architectural shift from TUI to a React 18 + TypeScript web application with FastAPI backend, featuring comprehensive infrastructure and real-time communication.

**Hybrid Routing & Multi-Provider Support (v0.3.3-0.4.0)**: Implementation of intelligent LLM routing between local Ollama models and remote OpenAI providers, with comprehensive provider management and advanced BMAD features.

**Advanced Agent Systems (v0.4.1-0.5.0)**: Integration of enhanced benchmarking capabilities, stability fixes, and communication improvements with autonomous mode optimizations.

**BMAD Pipeline Automation (v0.5.1-0.6.7)**: Complete automation of BMAD pipelines with infinite validation loops, resume functionality, and advanced processing capabilities.

**Research & Collaboration Enhancements (v0.2.0-0.2.1)**: Extended research modes with assisted and autonomous workflows, enhanced chat exchange capabilities for multi-user conversations, and improved collaborative research tools.

**Cyber Security & Monitoring (v0.2.2+)**: AI-powered threat analysis integration, comprehensive security monitoring, and advanced external benchmarking against industry standards.

**Cyber Security & Monitoring (v0.0.21+)**: AI-powered threat analysis integration, comprehensive security monitoring, and advanced external benchmarking against industry standards.

The current version represents a mature, enterprise-ready multi-agent orchestrator with modern web interfaces, hybrid LLM routing, and comprehensive research and development automation capabilities.

## 📋 Overview

Freya is a **BMAD-aligned multi-agent orchestrator** for local LLMs, featuring hybrid routing, real-time collaboration, and comprehensive benchmarking. This repository contains the complete evolution of Freya from its initial release to the current version.

### 🎯 Key Features

- **Multi-Agent Orchestration**: 7 specialized agents in coordinated BMAD pipelines
- **Hybrid LLM Routing**: Intelligent selection between local (Ollama) and remote (OpenAI) models
- **Real-Time Chat**: Multiple conversation modes with autonomous operation
- **Comprehensive Benchmarking**: Performance testing with external benchmarks
- **Modern Web Interface**: React 18 + TypeScript with real-time updates
- **Cyber Security Integration**: AI-powered threat analysis and monitoring

cd freya
pip install -r requirements.txt
cd web && npm install && cd ..

# Start the application

python -m freya
